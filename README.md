# Goofy Soundboard
Goofy Soundboard to aplikacja mobilna, która odtwarza memiczne dźwięki.<br>
Aplikacja ta została napisana w środowisu xamarin.forms na android.<br>
Do odtwarzania dźwięków używa wtyczki SimpleAudioPlayer.<br>
